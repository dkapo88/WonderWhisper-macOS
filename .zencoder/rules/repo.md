# WonderWhisper Mac Information

## Summary
WonderWhisper Mac is a SwiftUI-based macOS application for audio transcription and dictation. It provides real-time speech-to-text capabilities with multiple transcription provider integrations (AssemblyAI, Deepgram, Groq, Parakeet, and native Apple transcription). The app features a menu bar interface, waveform visualization, and integration with various LLM providers for processing transcribed text.

## Structure
- `WonderWhisper Mac/`: Main application source files containing SwiftUI views, audio processing, transcription providers, and LLM integrations
- `WonderWhisper MacTests/`: Unit tests for core functionality
- `WonderWhisper MacUITests/`: UI tests for application flows
- `DerivedData_WW/`: Local build artifacts and derived data
- `build/`: Build outputs generated by xcodebuild

## Language & Runtime
**Language**: Swift
**Framework**: SwiftUI
**Platform**: macOS
**Build System**: Xcode
**Package Manager**: Swift Package Manager

## Dependencies
**Main Dependencies**:
- FluidAudio (v0.4.0): Audio processing library

## Build & Installation
```bash
# Open project in Xcode
open "WonderWhisper Mac.xcodeproj"

# Build from command line
xcodebuild -project "WonderWhisper Mac.xcodeproj" -scheme "WonderWhisper Mac" -configuration Debug build

# Run the app from command line
open build/Debug/WonderWhisper\ Mac.app
```

## App Permissions
**Entitlements**:
- Audio input access
- Network client capabilities
- User-selected file read access
- Apple events automation (for AppleScript integration)
- App sandbox enabled

## Main Components
**Audio Processing**:
- `AudioRecorder.swift`: Handles audio recording
- `AudioPreprocessor.swift`: Processes audio for transcription
- `AudioDeviceManager.swift`: Manages audio input devices

**Transcription Services**:
- `AssemblyAIStreamingProvider.swift`: Integration with AssemblyAI
- `DeepgramStreamingProvider.swift`: Integration with Deepgram
- `GroqTranscriptionProvider.swift`: Integration with Groq
- `ParakeetTranscriptionProvider.swift`: Integration with Parakeet
- `NativeAppleTranscriptionProvider.swift`: Uses Apple's native transcription

**LLM Integration**:
- `GroqLLMProvider.swift`: Integration with Groq LLM
- `CerebrasLLMProvider.swift`: Integration with Cerebras LLM
- `OpenRouterLLMProvider.swift`: Integration with OpenRouter

**UI Components**:
- `ContentView.swift`: Main application view
- `SettingsView.swift`: Application settings
- `MenuBarController.swift`: Controls menu bar interface
- `WaveformOverlayController.swift`: Displays audio waveform visualization
